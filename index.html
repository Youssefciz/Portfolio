Projects

Enterprise Global Network Scanning & Vulnerability Monitoring System

Technologies: JavaScript, TypeScript, Python, NestJS, Prisma, PostgreSQL, Node.js, Raspberry Pi, Masscan, OpenVAS, Tailscale, PagerDuty

I built an enterprise-scale vulnerability monitoring platform designed to secure a distributed global infrastructure. The system runs lightweight Raspberry Pi agents at gallery locations worldwide, all managed through a centralized NestJS/Prisma backend with secure RBAC authentication and PostgreSQL persistence.

Agents communicate securely over Tailscale private networks, automatically discover local networks using masscan, and extract different /24 subnets from each office network. Each /24 subnet is then scanned individually with OpenVAS for comprehensive vulnerability assessment. The system supports incremental result saving for long-running scans, scaling reliably to 50k+ IPs per agent. A dedicated web UI enables intuitive scan scheduling and recurring automation, while PagerDuty integration provides enterprise level alerting for vulnerabilities, agent downtime, and infrastructure changes.

This project gave me hands on experience delivering a robust, enterprise ready distributed security solution, combining secure networking, scalable architecture, and automated vulnerability management.
